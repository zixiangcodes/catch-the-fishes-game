## Prompt 4

Okay, great so far!
Next, time to do the Javascript code and interactivity.

What needs to be done for each component.

1. Start Game Button
When players press the start game button, after 2 seconds, it will cause a sound wav file (music_background.wav) to play on loop and will cause the timer to start from 30 seconds to countdown to 0 seconds. 

2. Playing The Game
During the countdown from 30 seconds to 0 seconds:
- The fish images will change to random images (after intervals of 4 seconds) before the images are randomized again, from a list of 12 fish images which are:
>  Fish1_Close.png
>  Fish1_Open.png
>  Fish2_Close.png
>  Fish2_Open.png
>  Fish3_Close.png
>  Fish3_Open.png
>  Fish4_Close.png
>  Fish4_Open.png
>  Fish5_Close.png
>  Fish5_Open.png
>  Fish6_Close.png
>  Fish6_Open.png
- When players press any image that has the file names "Fish1_Open.png","Fish2_Open.png","Fish3_Open.png","Fish4_Open.png","Fish5_Open.png" or "Fish6_Open.png", a sound wav file (fish_open.wav) is played and the score increases by 1 point.
- When players press any image ending has the file names "Fish1_Close.png", "Fish2_Close.png","Fish3_Close.png","Fish4_Close.png","Fish5_Close.png" or "Fish6_Close.png", a sound wav file (fish_close.wav) is played and but the score doesn't increase by 1.
- Once the timer reaches 0, a message will display to tell players that the game is over and this will stop the sound wav file (music_background.wav) from playing and stop the loop.
- Then, another sound wav (game_finish.wav) file is played.

3. Game Ends
Then depending on the score obtained by the player, there will be different messages displayed and different sound wav files triggered.
- If a player gets a score of 0-9, the music_bad.wav plays and a message is displayed that says "Oh no, try again!"
- If a player gets a score of 10-20, the music_okay.wav plays and a message is displayed that says "Getting better!!"
- If a player gets a score of 21-40, the music_good.wav plays and a message is displayed that says "You're soo good now!!"
- If a player gets a score of 41 and above, the music_best.wav plays and a message is displayed that says "YOU'RE TOO AMAZING!! THANKS FOR PLAYING!!"

4.  Reset Game button
When a player presses the "Reset Game" button, all previous sounds and musics are stopped, even the one on loop. Next, a sound wav (game_reset.wav) file is played, the end game and game over messages are removed, before the page is refreshed.
